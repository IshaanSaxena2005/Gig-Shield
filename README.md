# Gig-Shield

A comprehensive insurance platform for gig workers, featuring weather-based risk assessment, automated claims processing, and AI-powered fraud detection.

## Features

- **Worker Dashboard**: Manage policies, view claims, and track premiums
- **Admin Dashboard**: Oversee policies, claims, and risk zones
- **AI Risk Assessment**: Weather-based premium calculation
- **Automated Claims**: Trigger-based claim processing
- **Fraud Detection**: ML-powered anomaly detection

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **AI Engine**: Python, scikit-learn
- **Deployment**: Docker, Kubernetes

## Getting Started

1. Clone the repository
2. Install dependencies for frontend and backend
3. Set up environment variables
4. Run the development servers

## Project Structure

- `frontend/` - React application
- `backend/` - Node.js API server
- `ai-engine/` - Python ML models
- `scripts/` - Utility scripts
- `docs/` - Documentation
